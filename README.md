Pastee
======

simple tool for store & share text


Requirements
------------

- perl
	- plack
	- carton
- node.js
	- grunt.js
	- bower
- sqlite


Install
-------

    $ git clone https://github.com/munky69rock/Pastee.git
	$ cd Pastee
	$ npm install
	$ grunt bower:install
	$ carton install
	$ carton exec plackup app.psgi
